Инструкция по установке

  1. Скачайте папку с локализацией по ссылке выше.
  2. Первый файл в папке перекиньте в EXBO\runtime\stalcraft\modassets\assets\stalker\lang, подтверждая замену файлов.
  3. Второй файл в папке перекиньте в EXBO\runtime\stalcraft\modassets\assets\gloomycore\lang, подтверждая замену файлов.
  4. Запустите игру, выбрав русский язык в настройках.
  5. Наслаждайтесь игрой!
